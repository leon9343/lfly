Detailed Description

Manages the asynchronous logging system, handling the message queue and the various Logger instances.

The LoggerManager is implemented as a singleton to ensure only one instance handles the logging process. It enqueues logs and processes them asynchronously in a separate thread to not impact performance for the main application
